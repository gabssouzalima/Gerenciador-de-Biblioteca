# Sistema de Biblioteca em JavaScript

## Descrição do Projeto
Este projeto implementa um **Sistema de Biblioteca** utilizando **Programação Orientada a Objetos (POO) em JavaScript**. O sistema permite gerenciar livros e usuários, registrar empréstimos e devoluções, e acompanhar a disponibilidade dos livros.  

O projeto evoluiu para uma interface **interativa**, utilizando **DOM** para manipulação dinâmica dos dados e **Tailwind CSS** para estilização, padronização de estilos e responsividade.

---

## Funcionalidades

- Cadastro de livros e usuários
- Empréstimo e devolução de livros
- Controle de disponibilidade dos livros
- Relacionamento entre livros e usuários
- Interface interativa usando DOM
- Layout responsivo e estilizado com Tailwind CSS
- Validações para evitar operações inválidas

---

## Tecnologias Utilizadas

- **JavaScript**: lógica e POO
- **HTML**: estrutura da interface
- **CSS / Tailwind CSS**: estilização e responsividade
- **DOM**: manipulação dinâmica da interface
